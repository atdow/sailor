# Sailor

水手低码平台（Sailor Low-Code Platform）

基于[amis-editor-demo](http://aisuda.github.io/amis-editor-demo)改造

技术栈：

-   服务端：Koa + MongoDB
-   前端：React + Amis

![](./architecture.png)

## 本地运行

1. `npm i` 安装依赖
2. `cd server` 进入服务端目录，再 `npm i` 安装服务端依赖
3. `npm server:start` 跑后端服务
4. `npm run dev` 编译前端
5. `npm run client:start` 启动前端，http://localhost:8082/
